# User Management System
Developed a Todo Application with JSP, Servlets, JDBC, and MySQL, featuring CRUD operations and user authentication.

## Following features
- Add todo
- Edit todo
- List Dodo
- Delete Todo
- Login
- Signup
- Responsive layout

## Technology stack
- Java 
- JSP and Servlet
- JDBC
- MySQL database
- Eclipse IDE
- Tomcat server
